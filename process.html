How to puch your work to GitHub

  • first create a repo on github. you can also get your pat(personal access token) ready too
    • How to generate the token

After generating the token

Open your terminal in your vs code, type and follow the steps below

  1. git init
  2. git status
  3. git add .
  4. git status
  5. git commit 'first commit'
  6. git config --global user.email "Github email account"
  7. git config --global user.name "Github account username"
  8. git config --global user.password "past the token you copy and save here"
  9. git commit -m 'first commit'
  10. git remote add origin "copy your respostory url and past here"
  11. git branch
  12. git push origin branchName
  13. Note this will give you error
  14. Do this: locate (https://github.com/ugonnachidinmac/yt2_git_push1/security/secret-scanning/unblock-secret/2iZOtBKUQOHlAlvvn47Ogj6vAA8) in the error massage
  15. ctrl + click the link (https://github.com/ugonnachidinmac/yt2_git_push1/security/secret-scanning/unblock-secret/2iZOtBKUQOHlAlvvn47Ogj6vAA8)
  16. login in to your github account
  17. select to test
  18. come back to terminal
  19. git push origin branchName
  20. go to your github account and refrensh the page
