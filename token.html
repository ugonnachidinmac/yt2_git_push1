How to generate Token

In Git and GitHub, a token is typically used for authentication and authorization.


By using tokens effectively and securely, you can automate and streamline many of your Git and GitHub workflows.

Steps on how to generate a token

  1. Click on the Profile and Setting Menu on the right hand of the page
  2. Click on setting
  3. Scrol down, from the left select Developer Setting
  4. Click on Personal Access tokens
  5. Choose token (classic)
  6. Click on Generate new tokens
  7. Select Generate new token (classic)
  8. Type in your Github Account Password and comfirm it
  9. Type in any note
  10. Select the Expiration duration (e.g 30days, 60days etc)
  11. Select the scopes
  12. Click on Genrate tokens
  13. Copy the token to Ms Word or Notebook to save
  14. Go back to Profile and setting menu and selecte your profile
  15. Select the repository your are working on
